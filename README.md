# csda-fire-product

A self-contained PowerPoint slide: **fire perimeter map** (CAL FIRE) + **5 satellite images** (Planet, Satellogic, Umbra, ICEYE, Landsat) for the same or closest post-fire date.

- **Map**: QGIS (perimeter + CA/US inset).
- **Imagery**: [CSDAP](https://csdap.earthdata.nasa.gov/explore/) + Google Earth Engine (Landsat).
- **Fire data**: [California Fire Perimeters (all)](https://gis.data.ca.gov/datasets/CALFIRE-Forestry::california-fire-perimeters-all).

See **[PROJECT_PLAN.md](PROJECT_PLAN.md)** for goals, workflow, and top-down vs bottom-up approach. Use **`data/study_area_tracker.csv`** to record candidate fires and image IDs per source once you have access.
